# api.social.com

## 开发环境：

* java8
* tomcat 9.0.100

## 项目结构：

```
api.example.com/
├── src/
│   ├── main/
│   │   ├── java/          # Java源代码目录
│   │   ├── resources/     # 配置文件目录
│   │   ├── sql/          # SQL脚本目录
│   │   └── webapp/       # Web应用资源目录
├── pom.xml               # Maven项目配置文件
└── target/              # 编译输出目录
```
